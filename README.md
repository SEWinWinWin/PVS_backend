### Run App Step
1. set java home:  https://javatutorial.net/set-java-home-windows-10
2. open cmd window
3. cd ./target
4. java -jar springboot-1.0-SNAPSHOT.jar
5. open Postman
6. test Get request : http://localhost:8080/products/1
